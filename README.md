# Sign Language Detection (A–Z, 0–9)

A real-time sign language recognition system built by **Anshuman Mathur**.  
It uses deep learning and a webcam to detect hand gestures and convert them to text.

### Key Features
- Real-time hand sign detection
- Supports A–Z and 0–9
- Built using CNN (MobileNetV2) + OpenCV + MediaPipe
- Fine-tuned with my own dataset for improved accuracy
